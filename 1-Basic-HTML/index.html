Community Computing PPP Projects Requests

Community Computing is a PPP class. In it, we use our knowledge of coding and computer science to serve the community.

To prepare for this work, we’ll learn some exciting new tools and concepts. We will learn the Python language, with a focus on using algorithms to sort and manipulate data. We’ll look at how programming languages interface with each other in real world applications.

This year, we will be making projects for the community that are based primarily around programming.
